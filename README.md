## 🔨 What I'm Currently Building

### 📲 WhatsApp AI Bot
An intelligent WhatsApp Business API bot built with Node.js and Claude AI — featuring auto-replies, keyword matching, slash commands, and human handoff logic. Real conversations, real automation.

### 📞 Twilio Call Centre System
A full outbound call centre platform targeting African markets (🇳🇬 Nigeria · 🇰🇪 Kenya · 🇿🇦 South Africa) — complete with IVR, call recording, agent dashboard, and SMS support. Built from scratch.
